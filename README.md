# AWL Tool Workflows

This repository contains JSON workflow files for the **Agentic Web Learning Tool (AWL Tool)** Chrome extension. These workflows can be imported into the extension to automate various web-based tasks.

## Available Workflows

- **[Iâ€™m feeling lucky!](i-m-feeling-lucky.json)**: Injects Google logos into the current page.
- **[Let It Snow!](let-it-snow.json)**: Injects a snowfall effect into the current page.
- **[Make It Rain](make-it-rain.json)**: Injects a rain effect into the current page.
- **[Reddit Sentiment Guard (Reasoned + Reveal)](reddit-sentiment-guard.json)**: Censors negative comments with a reason, but allows clicking to reveal.
- **[Translator](translate-to-spanish.json)**: Translates the given Wikipedia page from English to Spanish.

## How to Use

1. Open the AWL Tool Chrome extension.
2. Locate the import functionality within the extension's options page's Workflow section.
3. Select one of the JSON files from this repository to load the workflow.
